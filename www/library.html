Z88DK - Library

The library supplied with z88dk is quite large and supports a large number of features. As such it has been cut down into a number of sections which are:

  • Standard Library - Documentation of the standard C Library
  • Games Library - Detailing the library which is designed to make cross platform games programming possible
  • Z88 Library - Details of the extra routines which are available only on the z88 platform

More documentation is planned, and what we have we do plan to improve and supplement.
